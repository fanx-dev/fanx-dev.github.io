class vaseMath::Transform2D

sys::Obj
  vaseMath::Transform2D

@Js

AffineTransform. This matrix is a transpose of java.awt.geom.AffineTransform.

m00 m10 0
(x1, y1, 1) = (x y 1)  *   m01 m11 0
                           m02 m12 1

But the awt/swt Transform like this:

[ x']   [  m00  m01  m02  ] [ x ]   [ m00x + m01y + m02 ]
[ y'] = [  m10  m11  m12  ] [ y ] = [ m10x + m11y + m12 ]
[ 1 ]   [   0    0    1   ] [ 1 ]   [         1         ]

Don't like Transform2D, The Transform3D using the second way.
