sys::Obj
  compilerx::CommonType
common

static CType common(CNamespace ns, List<CType> types)

Given a list of types, compute the most specific type which they all share, or at worst return sys::Obj. This method does not take into account mixins, only extends class inheritance.

commonType

static CType commonType(CNamespace ns, List<Expr> exprs)

Given a list of Expr instances, find the common base type they all share. This method does not take into account the null literal. It is used for type inference for lists and maps.
